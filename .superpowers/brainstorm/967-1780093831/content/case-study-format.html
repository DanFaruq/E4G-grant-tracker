A — Story Scroll

One long page per case study. Sections animate in as you scroll: Problem → Approach → What I built → Result. Feels editorial, like a magazine feature. Immersive and impressive.

B — Tabbed Deep-Dive

Each case study is a compact card that expands into tabs: Overview / Problem / Solution / Tech Stack / Results / Live Demo. Client can jump to what matters to them. Efficient and scannable.

C — App Walkthrough

Annotated carousel of actual app screenshots. Each screen has a hotspot label explaining what it does and why it matters. Ends with a "View Live Demo" button. Shows the real product, not just words about it.

You can combine approaches — e.g. Story Scroll with an App Walkthrough embedded inside it.
